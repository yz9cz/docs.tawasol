عرض وقت التنفيذ

جديد

أصبحت الاستجابة تعرض وقت تنفيذ الطلب مع الرد

تم إضافة معلومات دقيقة عن وقت التنفيذ في جميع الاستجابات لمساعدتك في مراقبة الأداء وتحسين التطبيقات.

الحقول الجديدة:

  • execution_time_seconds - وقت التنفيذ بالثواني (3 أرقام عشرية)
  • execution_time_ms - وقت التنفيذ بالميلي ثانية (رقم عشري واحد)

مثال على الاستجابة الجديدة:

{
  "id": 587193,
  "amount": "38000",
  "number": "6589536",
  "status": 1,
  "message": "تم الدفع بنجاح",
  "execution_time_seconds": 2.456,
  "execution_time_ms": 2456.0
}

نموذج الإدخال

JSON Schema

  "id": 587193,
  "amount": "38000",

  "code": "",

  "type": "1 MB",
  "date": "2025-08-07T16:06:33.000000Z"
}

معاملات الإدخال

المعامل النوع مطلوب الوصف
id integer ✅ معرف الطلب الفريد
amount string ✅ مبلغ الدفع
number string ✅ رقم الطلب
code string ❌ رمز المحافظة (يمكن أن يكون فارغ)
product integer ✅ معرف المنتج (50, 32, 47, 45, 148, 46, 29, 30, 60, 70)
type string ✅ نوع المنتج
date string ✅ تاريخ الطلب بصيغة ISO

رموز الحالة

1
تم الدفع بنجاح

تم معالجة الدفع بنجاح

2
خطأ الرقم أو الشركة

الرقم غير صحيح أو الشركة غير مدعومة

3
مشكلة تقنية

حدث خطأ تقني أثناء المعالجة

4
يوجد دين على المشترك

لا يمكن إتمام العملية بسبب وجود دين
