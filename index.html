نموذج الإدخال

JSON Schema 
{
  "id": 587193,
  "amount": "38000",
  "number": "6589536",
  "code": "",
  "product": 50,
  "type": "1 MB",
  "date": "2025-08-07T16:06:33.000000Z"
}

معاملات الإدخال

المعامل النوع مطلوب الوصف
id integer ✅ معرف الطلب الفريد
amount string ✅ مبلغ الدفع
number string ✅ رقم الطلب
code string ❌ رمز المحافظة (يمكن أن يكون فارغ)
product integer ✅ معرف المنتج (50, 32, 47, 45, 148, 46, 33, 29, 30, 60)
type string ✅ نوع المنتج
date string ✅ تاريخ الطلب بصيغة ISO

رموز الحالة

1
تم الدفع بنجاح

تم معالجة الدفع بنجاح

2
خطأ الرقم أو الشركة

الرقم المدخل غير صحيح أو غير موجود في قاعدة بيانات الشركة

3
فشل في عملية الدفع

حدث خطأ أثناء المعالجة أو تكرار في الدفع أو مشكلة تقنية

4
المبلغ المطلوب أكبر

المبلغ المدفوع غير كافي - يتم إرسال المبلغ المطلوب في الرسالة
